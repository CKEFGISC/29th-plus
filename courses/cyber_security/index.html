資安

介紹

你對駭客的認知基本上都是錯的—駭客不會整天穿帽 T、輸入幾行 code 就能發射核彈、打字很快。

資安小社會讓你建立對駭客的專業領域—資訊安全的基本認識。講師從 Linux 講起，一路帶到網路、CTF 這些資安的基礎。

這堂課不會讓你變成駭客，但絕對能讓你充分認識資安這個領域。
